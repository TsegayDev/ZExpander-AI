# **ZExpander-AI**

**Next.js AI app using the Gemini API for sentence expansion, tone analysis, and document summarization.**

---

## **Overview**

ZExpander-AI is an intelligent web application built with **Next.js** that helps users enhance their writing and analyze text effortlessly. It leverages AI capabilities to **expand sentences**, **analyze the tone of text**, and **summarize documents**, providing actionable insights in real-time.

The app integrates with the **Gemini API** to power its AI-driven functionalities, making it a versatile tool for writers, students, and professionals who want to improve text clarity, style, and readability.

---

## **Key Features**

* **Sentence Expansion**: Automatically rephrase or expand sentences to make writing more expressive.
* **Tone Analysis**: Detect the emotional tone of text (e.g., formal, friendly, neutral) to ensure the right communication style.
* **Document Summarization**: Summarize long text or documents into concise key points.
* **User-Friendly Interface**: Built with modern UI/UX principles using Next.js and React.
* **Modular & Scalable**: Designed for easy expansion and integration with other AI tools.

---

## **Tech Stack**

* **Frontend & Framework**: Next.js, React
* **AI Integration**: Gemini API
* **Styling**: Tailwind CSS
* **Deployment Ready**: Optimized for modern web hosting platforms
